Spanish (Español)

Para instalar tu personaje siga los pasos:
- ingrese a la carpeta character_data y revise el archivo README que se encuentra alli.
- Ingrese a character_installer y modifique la clase ModInfo

class ModInfo:
	mod_dir = 'character_data'  # folder (es el nombre de la carpeta)
	mod_name = 'character'  # name (nombre del mod)
	creator = 'SEBASTIAN2059' # creator (quien lo creo)
	url = '' # video (¿subiste un video a YT?)

Una vez hayas terminado coloca la carpeta y el archivo en tu carpeta de mods (Puedes eliminar los README)

Nota: Si instalas esta plantilla sin seguir los pasos porbablemente no funcione o cause errores.

- Enter character_installer and modify the ModInfo class

class ModInfo:
mod_dir = 'character_data' # folder (is the name of the folder)
mod_name = 'character' # name (mod name)
